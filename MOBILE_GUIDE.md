# 📱 SignPush - Руководство по мобильной версии

## 🎯 Обзор

SignPush полностью адаптирован для мобильных устройств с оптимизацией для:
- 📱 **Смартфонов** (iPhone, Android)
- 📱 **Планшетов** (iPad, Android tablets)
- ✋ **Сенсорного управления** (touch gestures)
- 🚀 **PWA** (Progressive Web App) - возможность установки как приложение

---

## ✨ Основные возможности на мобильных

### **1. Адаптивный дизайн**
- ✅ Автоматическая подстройка под размер экрана
- ✅ Вертикальная и горизонтальная ориентация
- ✅ Оптимизация для экранов от 320px до 1024px

### **2. Сенсорное управление**
- ✅ Рисование подписи пальцем на canvas
- ✅ Увеличенные кнопки (минимум 44x44px по стандартам Apple)
- ✅ Плавная прокрутка и зум PDF документов
- ✅ Жесты pinch-to-zoom для PDF

### **3. Установка как PWA**
- ✅ Установка на главный экран телефона
- ✅ Работа в полноэкранном режиме (без адресной строки браузера)
- ✅ Иконка приложения на домашнем экране
- ✅ Быстрые действия (shortcuts) из иконки

### **4. Оптимизация производительности**
- ✅ Быстрая загрузка на мобильном интернете
- ✅ Минимальное использование батареи
- ✅ Плавные анимации (60 FPS)
- ✅ Оптимизированные изображения

## 📐 Breakpoints (точки адаптации)

### **Desktop (> 1024px)**
- Полная версия с широкими таблицами
- Все колонки видны
- Большие preview документов

### **Планшет (768px - 1024px)**
```css
@media (max-width: 1024px) {
  /* Уменьшенные отступы */
  /* Адаптированные таблицы */
  /* Компактный toolbar */
}
```

### **Смартфон (до 768px)**
```css
@media (max-width: 768px) {
  /* Скрыты второстепенные колонки (Hash, Размер) */
  /* Таблица в карточном виде */
  /* Вертикальный layout кнопок */

}
```

### **Маленькие смартфоны (до 480px)**
```css
@media (max-width: 480px) {
  /* Таблица полностью в режиме карточек */
  /* Компактные шрифты */
  /* Минимальные отступы */
}
```

## 🎨 Адаптивные компоненты

### **1. Таблица документов**

**Desktop вид:**
```
┌──────────────┬────────┬──────────┬────────┬──────────┐
│ Документ     │ Hash   │ Дата     │ Размер │ Действия │
├──────────────┼────────┼──────────┼────────┼──────────┤
│ contract.pdf │ a3f5d8 │ 21.02.26 │ 1.2 MB │ [Кнопки] │

```

**Мобильный вид (карточки на экранах < 480px):**
```

│ 📄 Документ: contract.pdf           │
│ 📅 Дата создания: 21.02.26         │
│                                     │
│ [👁️ Просмотр]  [✍️ Подпись]       │
└─────────────────────────────────────┘
```

### **2. Модальное окно подписи**

**Адаптация для мобильных:**
- Полноэкранный режим (fullscreen)
- Canvas занимает всю доступную высоту
- Толще линии границы для лучшей видимости
- `touch-action: none` для предотвращения случайной прокрутки
- Увеличенные кнопки "Отменить" и "Добавить"

**Размеры canvas:**
```javascript
// Desktop
height: 400px

// Планшет
height: 350px

// Смартфон (портрет)
height: calc(100vh - 200px)

// Смартфон (ландшафт)
height: calc(100vh - 140px)
```

### **3. PDF Viewer**

**Мобильные оптимизации:**
```javascript
// Адаптивный canvas
.pdf-content canvas {
  max-width: 100%;
  height: auto !important;
  touch-action: pan-x pan-y pinch-zoom; // Жесты масштабирования
}
```

**Toolbar:**
- Вертикальное расположение на узких экранах
- Кнопка "Сохранить подпись" на всю ширину
- Индикатор загрузки (spinner) при сохранении

## 📲 Установка PWA на устройства

### **iOS (iPhone/iPad)**

1. Откройте SignPush в Safari
2. Нажмите кнопку **"Поделиться"** (квадрат со стрелкой вверх)
3. Прокрутите вниз и выберите **"На экран «Домой»"**
4. Нажмите **"Добавить"**

✅ Иконка SignPush появится на домашнем экране!

**Особенности iOS:**
- Полноэкранный режим без адресной строки
- Нативная анимация открытия
- Работает как обычное приложение
- Цвет статус-бара соответствует дизайну

### **Android**

1. Откройте SignPush в Chrome
2. Появится всплывающее окно **"Установить приложение"**
3. Нажмите **"Установить"**

**Или вручную:**
1. Нажмите меню (три точки) ⋮
2. Выберите **"Установить приложение"** или **"Добавить на главный экран"**

✅ Иконка SignPush добавлена в список приложений!

**Особенности Android:**
- Полноценное приложение в списке Apps
- Уведомления (если добавить в будущем)
- Тема оформления соответствует системной
- Быстрые действия при долгом нажатии на иконку

## 🎯 Использование на мобильных

### **1. Загрузка документа**

```
┌─────────────────────────────────┐
│                                 │
│    [📄 Выберите файл]          │
│    (кнопка на всю ширину)       │
│                                 │
└─────────────────────────────────┘
```

- Нажмите кнопку "Выберите файл"
- Откроется системный файловый менеджер
- Выберите PDF из галереи/файлов
- Документ автоматически загрузится

### **2. Просмотр документа**

**Портретная ориентация:**
- PDF занимает всю ширину экрана
- Прокрутка вертикальная
- Toolbar сверху (компактный)

**Ландшафтная ориентация:**
- PDF растягивается по ширине
- Toolbar минимизирован
- Больше места для контента

### **3. Добавление подписи**

**Шаг 1:** Нажмите кнопку с иконкой **пера** 🖊️

**Шаг 2:** Откроется полноэкранное окно для рисования
```
┌───────────────────────────────────┐
│  [✖️ Закрыть]   Место для подписи  │
├───────────────────────────────────┤
│                                   │
│     [Рисуйте пальцем здесь]      │
│                                   │
│         ╱╲                        │
│        ╱  ╲    Ваша подпись      │
│       ╱    ╲                      │
│                                   │
├───────────────────────────────────┤
│  [Отменить]        [Добавить]     │
└───────────────────────────────────┘
```

**Шаг 3:** Выделите область на PDF (касанием и перетаскиванием)

**Шаг 4:** Нажмите **"💾 Сохранить подпись"**
```
┌───────────────────────────────────┐
│  [💾 Сохранить подпись]           │
│  (зелёная кнопка на всю ширину)   │
└───────────────────────────────────┘
```

**Индикатор сохранения:**
```
┌───────────────────────────────────┐
│  [⏳ Сохранение...]               │
│  (с анимацией spinner)            │
└───────────────────────────────────┘
```

## 🔧 Технические особенности

### **Meta теги для мобильных**

```html
<!-- Viewport -->
<meta name="viewport" content="width=device-width, initial-scale=1, 
      maximum-scale=5, user-scalable=yes, viewport-fit=cover" />

<!-- iOS специфика -->
<meta name="apple-mobile-web-app-capable" content="yes" />
<meta name="apple-mobile-web-app-status-bar-style" content="default" />
<meta name="apple-mobile-web-app-title" content="SignPush" />

<!-- Android -->
<meta name="theme-color" content="#1B9AFF" />
<meta name="mobile-web-app-capable" content="yes" />

<!-- Отключение автоопределения номеров телефонов -->
<meta name="format-detection" content="telephone=no" />
```

### **Touch оптимизация CSS**

```css
/* Плавная прокрутка на iOS */
* {
  -webkit-overflow-scrolling: touch;
  -webkit-tap-highlight-color: rgba(0, 0, 0, 0.1);
}

/* Canvas подписи - отключение стандартных жестов */
.sigPad {
  touch-action: none;
  -webkit-user-select: none;
  user-select: none;
}

/* PDF с поддержкой масштабирования */
.pdf-content canvas {
  touch-action: pan-x pan-y pinch-zoom;
}
```

### **Минимальные размеры touch targets**

```css
/* По стандартам Apple - минимум 44x44px */
button, a, input[type="file"] {
  min-height: 44px;
  min-width: 44px;
}

/* Увеличенные кнопки на мобильных */
@media (max-width: 768px) {
  .btn {
    padding: 12px 20px;
    font-size: 16px;
  }
}
```

## 🐛 Известные ограничения

### **iOS Safari:**
- ⚠️ PWA не поддерживает push уведомления
- ⚠️ Ограниченное хранилище (50MB для LocalStorage)
- ✅ **Решение:** Используйте облачное хранилище

### **Старые Android (< 8.0):**
- ⚠️ Ограниченная поддержка CSS Grid
- ⚠️ Медленный canvas rendering
- ✅ **Решение:** Обновите браузер Chrome

### **Маленькие экраны (< 320px):**
- ⚠️ Очень мало места для UI
- ✅ **Решение:** Rotate device или используйте планшет

## 🎓 FAQ - Частые вопросы

### **Q: Можно ли работать офлайн?**
A: После первой загрузки - да! PWA кеширует основные файлы. Но для сохранения документов нужен интернет (запросы к backend).

### **Q: Поддерживается ли Apple Pencil?**
A: Да! Canvas работает с любым стилусом, включая Apple Pencil и S Pen.

### **Q: Безопасно ли на мобильных?**
A: Да, все данные передаются по HTTPS, подписи хешируются SHA-256, SQL-инъекции защищены.

### **Q: Можно ли использовать на планшете?**
A: Конечно! На планшетах (iPad, Galaxy Tab) интерфейс адаптируется и показывает больше информации.

### **Q: Сколько занимает места на телефоне?**
A: PWA занимает ~2-5MB (зависит от кеша). Документы хранятся на сервере.
