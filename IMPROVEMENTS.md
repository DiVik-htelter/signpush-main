## 🔒 Критические исправления безопасности

### **Проблема: SQL-инъекции**

**Было (ОПАСНО ⚠️):**
```python
# database.py - УЯЗВИМО К SQL-ИНЪЕКЦИЯМ
cursor.execute(f"SELECT password FROM users WHERE login = '{login}';")
cursor.execute(f"DELETE FROM documents WHERE id = '{id}';")
```

**Стало (БЕЗОПАСНО ✅):**
```python
# database.py - ПАРАМЕТРИЗОВАННЫЕ ЗАПРОСЫ
cursor.execute("SELECT password FROM users WHERE login = %s;", (login,))
cursor.execute("DELETE FROM documents WHERE id = %s;", (id,))
```

### Что это даёт:
- ❌ **Невозможно** выполнить произвольный SQL код
- ✅ Защита от атак типа `'; DROP TABLE users; --`
- ✅ Автоматическое экранирование специальных символов

## 🆕 Новая функциональность

### **1. Встраивание подписи в PDF**

#### Как это работает:

```
Пользователь                 Frontend                 Backend                 БД
    |                            |                        |                     |
    |--[1] Рисует подпись------->|                        |                     |
    |    (SignatureModal)        |                        |                     |
    |                            |                        |                     |
    |--[2] Размещает на PDF----->|                        |                     |
    |    (выделяет область)      |                        |                     |
    |                            |                        |                     |
    |--[3] Нажимает "Сохранить"->|                        |                     |
    |                            |                        |                     |
    |                            |--[4] POST /api/sign--->|                     |
    |                            |    document            |                     |
    |                            |    {document_id,       |                     |
    |                            |     signature_base64,  |                     |
    |                            |     x, y, width, h}    |                     |
    |                            |                        |                     |
    |                            |                        |--[5] Получить PDF-->|
    |                            |                        |<---документ---------|
    |                            |                        |                     |
    |                            |                        |--[6] Встроить------>|
    |                            |                        |    подпись в PDF    |
    |                            |                        |    (PyMuPDF/fitz)   |
    |                            |                        |                     |
    |                            |                        |--[7] Вычислить----->|
    |                            |                        |    новый SHA-256    |
    |                            |                        |                     |
    |                            |                        |--[8] Сохранить----->|
    |                            |                        |    подписанный PDF  |
    |                            |<---[9] Success---------|                     |
    |<--[10] Документ подписан---|                        |                     |
```

### **2. Модуль pdf_signer.py**

**Класс `PdfSigner`:**
```python
class PdfSigner:
    @staticmethod
    def add_signature_to_pdf(
        pdf_base64: str,           # Исходный PDF
        signature_base64: str,     # Изображение подписи
        page_number: int,          # Номер страницы
        x, y, width, height        # Позиция и размер
    ) -> Tuple[str, bool]:
        """
        Встраивает визуальную подпись в PDF.
        Возвращает: (подписанный_pdf_base64, успех)
        """
```

**Ключевые особенности:**
- 📄 Работает с PDF через PyMuPDF (fitz)
- 🖼️ Встраивает PNG/JPG изображение подписи
- 📐 Точное позиционирование с учетом масштаба canvas
- 🗜️ Автоматическое сжатие и оптимизация PDF
- ✅ Подпись становится частью PDF навсегда

### **3. API Endpoint: POST /api/sign-document**

**Request:**
```json
{
  "document_id": 123,
  "signature_base64": "data:image/png;base64,iVBORw0KGg...",
  "page_number": 0,
  "x": 100.5,
  "y": 200.3,
  "width": 150,
  "height": 75,
  "login": "user@example.com"
}
```

**Response (Success):**
```json
{
  "success": true,
  "message": "Документ успешно подписан и сохранён",
  "new_document_id": 456,
  "hash": "a3f5d8c9e2b1..."
}
```

**Response (Error):**
```json
{
  "success": false,
  "message": "Документ не найден"
}
```

## 🎮 Использование

### **1. Загрузка документа**
- Нажмите "Выберите файл"
- Выберите PDF документ
- Документ автоматически сохранится в БД

### **2. Просмотр документа**
- Нажмите "Просмотр" в таблице документов
- PDF откроется в PDF Reader

### **3. Добавление подписи**
- Нажмите кнопку с иконкой **пера** (🖊️)
- Нарисуйте подпись в модальном окне
- Нажмите "Добавить"
- Выделите область на PDF, где должна быть подпись (мышью)
- Подпись появится в выделенной области

### **4. Сохранение подписанного документа**
- Нажмите кнопку **"Сохранить подпись"** (зелёная кнопка с иконкой 💾)
- Дождитесь завершения процесса (появится индикатор "Сохранение...")
- После успеха появится уведомление с ID нового документа и хешем
- Страница автоматически перезагрузится
- Подписанный документ появится в списке с пометкой "(Подписан)"

### **5. Очистка подписи**
- Нажмите кнопку с иконкой **ластика** (🧹)
- Подпись исчезнет с canvas (можно начать заново)

## 🔧 Технические детали

### **Координаты и масштабирование**

**Проблема:** Canvas на фронтенде и PDF используют разные системы координат.

**Решение:**
```javascript
// Frontend (pdf-reader.js)
const viewport = page.getViewport({ scale: 2.5 }); // Canvas масштаб

// Backend (pdf_signer.py)
scale_factor = 2.5
pdf_x = canvas_x / scale_factor  // Конвертация координат
```

### **Вычисление хеша**

```python
# Backend (main.py)
import hashlib

# Получаем чистую base64 строку без data URI
signed_pdf_clean = signed_pdf.split('base64,')[-1]

# Вычисляем SHA-256
new_hash = hashlib.sha256(signed_pdf_clean.encode()).hexdigest()
```

### **Формат данных подписи**

```javascript
// Frontend → Backend
{
  document_id: 123,           // ID документа для подписания
  signature_base64: "data:image/png;base64,...", // Изображение
  page_number: 0,             // 0-indexed номер страницы
  x: 150.5,                   // Координата X в canvas пикселях
  y: 300.2,                   // Координата Y в canvas пикселях
  width: 200,                 // Ширина области подписи
  height: 100,                // Высота области подписи
  login: "user@example.com"   // Email пользователя
}
```
